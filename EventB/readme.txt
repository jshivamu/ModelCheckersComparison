The file RodinVersions.png shows versions of various components of the Rodin tool, which provides an IDE for Event-B programs.

The folder AltNSpeedControl has the project file and metadata for the Event-B project. 

The modelling paradigm of Event-B requires writing an Event with guard conditions for each possible state which has to be uniquely handled. As a result, the program becomes lengthy. Hence, only the Stall recovery is modelled (Climb requirement is omitted).

The file Event_B_V2.docx contains the prompts to the LLMs and their output.
