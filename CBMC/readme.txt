This folder contains experiments with CBMC.
The file main2_Bounded.c contains the program (largely generated by LLMs, and errors are manually corrected).

The file BatchFile.bat contains commands that were tried for model checking with CBMC. The file also shows specific command and options used to invoke CBMC.

The files 10iter.txt, 50iter.txt, and 100iter.txt contain the output of the CBMC tool for loop unwinding counts of 10, 50 and 100 respectively.

The CBMC_V1.docx file contains the LLM outputs.

The file BatchFile_Output.txt contains the console output.

As seen in BatchFile_Output.txt, the time taken by CBMC to conclude the bounded model checks increases exponentially with the unwinding counts. For 10 counts of unwinding it took about 1.23 seconds. On the same machine, it took 10 minutes and 29.15 seconds for 50 iterations and it took more than 6 hours for 100 iterations.
